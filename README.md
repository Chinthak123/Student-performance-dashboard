# Student-performance-dashboard
Student Performance Analysis
Uncovering the factors that shape academic success.

Hi there! Welcome to my project. As part of my data science journey, I analyzed a dataset of over 600 student records to understand what really drives exam grades. My goal wasn't just to look at the numbers, but to find actionable insights that could help schools support their students more effectively.

💡 Why this project matters
Data is just noise until you give it context. In this project, I wanted to answer the questions that matter most to educators and students:

Does parental background change the playing field?

Do test prep courses actually work?

Who is at risk of falling behind, and why?

🛠️ The Tech Stack
I used Python to turn raw data into a story:

Pandas: For cleaning the data and crunching the numbers.

Seaborn & Matplotlib: To visualize the trends.

Streamlit: To build an interactive dashboard so the school principal can see the insights at a glance.

📈 Key Findings
After diving into the data, a few things stood out:

Prep Matters: Test preparation is the single biggest predictor of higher scores.

The Gender Gap: Performance trends differ by subject, suggesting we need tailored instructional approaches.

The "At-Risk" Group: Students scoring below 50 often share similar profiles, meaning we can identify them early and provide help before it's too late.

🚀 How to explore it
If you want to poke around the code or run the dashboard yourself:

Clone this repo: git clone [your-repo-link]

Install the requirements: pip install -r requirements.txt

Launch the dashboard: streamlit run app.py
